2.获取Scheduler信息

参数名称 类型 描述
schedulerName String (scheduler)调度器名称
isShutdown boolean 是否关闭
isStarted boolean 是否启动
isInStandbyMode boolean 是否处于准备模式
jobCount int (job)作业数量
triggerCount int (trigger)触发器数量
triggerErrorCount int (trigger)触发器错误数量
triggerBlockedCount int (trigger)触发器堵塞数量
triggerPausedCount int (trigger)触发器暂停数量
schedulerContext json array key-value的自定义的schedulerContext参数 
url/api/schedulerInfo.json
                            {
                            "resultCode": 1,
                            "content": [
                                    {
                                    "schedulerName": "quartzScheduler",
                                    "isShutdown": false,
                                    "isStarted": false,
                                    "isInStandbyMode": true,
                                    "jobCount": 2,
                                    "triggerCount": 2,
                                    "triggerErrorCount": 0,
                                    "triggerBlockedCount": 0,
                                    "triggerPausedCount": 1,
                                    "schedulerContext": []
                                    }
                                ]
                            }

 

3.启动Scheduler

参数名称 类型 是否必填 描述
schedulerName String YES (scheduler)调度器名称 
"ok"
url/api/schedulerStart.json?schedulerName=quartzScheduler
                            {
                                "resultCode": 1,
                                "content": "ok"
                             }
